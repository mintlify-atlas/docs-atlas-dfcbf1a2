## What is RedQuanta MCP?

RedQuanta MCP is a [Model Context Protocol](https://modelcontextprotocol.io) server built for professional security teams. It bridges the gap between AI assistants and real penetration testing tools, letting you orchestrate complex security assessments through natural language while maintaining enterprise-grade security controls.

## Get started in 3 steps

<Steps>
  <Step title="Install and build">
    Clone the repository, install dependencies, and build the TypeScript project.

    ```bash
    git clone https://github.com/sc4rfurry/RedQuanta-MCP.git
    cd RedQuanta-MCP
    npm install && npm run build
    ```
  </Step>
  <Step title="Connect your AI assistant">
    Add RedQuanta MCP to your Claude Desktop or Cursor configuration to start using security tools through natural language.

    ```json claude_desktop_config.json
    {
      "mcpServers": {
        "redquanta-mcp": {
          "command": "node",
          "args": ["./dist/server.js"],
          "cwd": "/path/to/RedQuanta-MCP",
          "env": {
            "MCP_MODE": "stdio",
            "JAIL_ROOT": "/path/to/RedQuanta-MCP/vol"
          }
        }
      }
    }
    ```
  </Step>
  <Step title="Run your first scan">
    Ask your AI assistant to scan a target you're authorized to test. RedQuanta handles tool execution, argument sanitization, and result formatting automatically.

    ```
    "Scan 192.168.1.0/24 for live hosts and open ports"
    "Perform a web security assessment on https://example.com"
    "Run a full enumeration workflow on 10.0.0.1"
    ```
  </Step>
</Steps>
